# loss_aversion

## 策略描述

利用损失厌恶心理（Loss Aversion）— 人们对避免损失的动机强于获得收益。在标题中强调用户可能错过的东西或正在犯的错误。

## 适用场景

- 错误/陷阱类内容（mistakes, avoid, wrong）
- 对比类内容（vs, better than）
- 警告类内容（warning, danger, risk）
- Position 6-15（需要强烈情感触发）

## 核心技巧

1. 使用"避免"、"停止"、"不要"等否定词
2. 强调常见错误或误区
3. 暗示当前做法可能有害
4. 提供"更好的替代方案"

## 示例

**Before**: "How to Invest in Stocks"
**After**: "5 Stock Investing Mistakes That Cost Beginners Thousands"

**Before**: "Best Password Managers"
**After**: "Stop Using the Same Password — Here's Why (And What to Do)"

## Prompt 模板

在标题中：
- 使用"避免"、"停止"、"不要"、"错误"等词
- 暗示当前做法有风险
- 包含目标关键词

在描述中：
- 具体说明损失的后果
- 提供解决方案的预览
- 制造紧迫感

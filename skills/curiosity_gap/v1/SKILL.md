# curiosity_gap

## 策略描述

利用信息缺口（Curiosity Gap）激发点击欲望。在标题中暗示有价值的信息，但不完全揭示答案，让用户产生"我必须点进去看看"的心理。

## 适用场景

- 教程类内容（how-to, guide）
- 问题解答（why, what, when）
- 排名/列表类内容（top N, best）
- Position 4-10（有一定曝光但 CTR 不足）

## 核心技巧

1. 使用"秘密"、"真相"、"原因"等词汇
2. 提出问题但不在标题中回答
3. 暗示有意外发现或反直觉结论
4. 避免过度承诺（clickbait）

## 示例

**Before**: "Python List Comprehension Tutorial"
**After**: "Why Python Experts Prefer This List Trick (Not What You Think)"

**Before**: "Best Coffee Makers 2024"
**After**: "The #1 Coffee Maker Baristas Actually Use at Home"

## Prompt 模板

在标题中：
- 使用疑问句或暗示性陈述
- 包含目标关键词
- 暗示有独特见解或意外发现

在描述中：
- 进一步强化好奇心
- 提供一个具体细节作为"诱饵"
- 以行动号召结尾
